# Battle Bots

Battle Bots is a Java game where players design and build their own robots, then send them into battle against each other. The objective is to eliminate every opponent and be the last robot standing.

## Overview

This project is based on a competitive battle arena concept where strategy and design both play an important role. Each player creates a robot with its own characteristics, and the robots fight until only one survives.

## Features

- Player-designed battle robots
- Arena-style combat
- Elimination-based gameplay
- Winner is the last robot standing
- Developed in Java using object-oriented programming principles

## Built With

- Java

## Game Objective

Design a robot, battle against opponents, and outlast everyone else to win the game.
